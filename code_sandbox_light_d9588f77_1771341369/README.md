# 具体⇄抽象トレーニングアプリ

楽しく学んで、会話力・説明力・問題解決力をアップさせるWebアプリケーションです。

## 🎯 アプリの目的

具体と抽象の行き来ができるようになるためのトレーニングアプリ。仕事やプライベートでのコミュニケーション能力、問題解決能力、アイデア創出力を向上させます。

## ✨ 実装済み機能

### 1. 具体抽象化ゲーム 🎮

#### 基本機能
- **参加人数**: 2〜20人まで選択可能
- **ターン数**: 1〜10ターンまで選択可能（各プレイヤーが指定したターン数だけ回答）
- **3つの難易度レベル**:
  - **Easy**: 2つの単語から共通点を見つける（得点倍率 ×1.0）
  - **Hard**: 3つの単語から共通点を見つける（得点倍率 ×1.5）
  - **Expert**: 4つの単語から共通点を見つける（得点倍率 ×2.0）

#### ゲームの流れ
1. 参加人数、ターン数、難易度を選択
2. 各プレイヤーの名前を入力（任意）
3. 「〇〇の番です」と表示され、ランダムに単語が表示される
4. 回答者が共通点を抽象化して回答
5. 回答者以外の全員が0〜5点（星マーク）で評価
6. 全プレイヤーが指定したターン数だけ回答
7. 全ターン終了後、総合得点を表示
8. 優勝者を決定、ハイスコアを保存

#### 単語データベース
**合計1,300語以上**の日本語単語を10カテゴリに分類：
- 感情・心理（100語）
- 自然・環境（100語）
- 人間・社会（100語）
- 抽象概念（100語）
- 物質・物体（100語）
- 文化・芸術（100語）
- 学問・知識（100語）
- 経済・ビジネス（100語）
- 日常生活（100語）
- 動物・植物（200語）
- 行動（100語）
- 状態（100語）

#### ハイスコアシステム
- 上位10件のハイスコアを保存・表示
- プレイヤー名、得点、難易度、人数、日付を記録
- ホーム画面で確認可能

### 2. ブレインストーミング機能 💡

#### 機能内容
- **テーマ入力欄**: ブレストのテーマを設定
- **付箋カード追加**: 「具体」「抽象」の2種類を追加可能
- **ドラッグ&ドロップ**: 付箋を自由に配置
- **色分け**: 具体（青系）、抽象（赤系）で視覚的に区別
- **編集機能**: 各付箋の内容を直接編集可能
- **削除機能**: 不要な付箋を個別に削除
- **保存・読込**: LocalStorageでブレスト内容を保存・読込
- **全クリア**: 全ての付箋を一括削除

### 3. 充実した解説ページ 📚

#### 解説内容（修正済み）
1. **なぜ具体⇄抽象が重要なのか？**
   - 会話が伝わらない原因
   - 仕事での問題解決に必須のスキル
   - AI時代に残れる会話力・説明力

2. **具体と抽象とは？**
   - 定義と特徴
   - 階層図での説明（生物→動物→哺乳類→人間→日本人）
   - エネルギーの分類例

3. **会話での使い分け**
   - 新人とベテランへの説明の違い
   - 中華料理の例
   - 反転した場合のトラブル例

4. **問題解決での活用**
   - 「勉強が苦手」の分解例
   - 原因→要因→対策の具体化
   - 数値化の重要性

5. **説明下手の解消方法**
   - 3歳の子供への仕事説明
   - HSAB則の日常語変換

6. **具体病と抽象病**
   - 具体病：本質を見失う
   - 抽象病：行動に落とし込めない

7. **学びを加速するトレーニング**
   - How（どうやって）とWhy（なぜ）を問いかける
   - 真反対・無関係なことを同じ抽象にまとめる
   - **抽象化トレーニング④：学んだ内容をXで投稿！**
     - ✅ **修正済み**: 「SNS」から「X」に変更
     - ✅ **追加説明**: 「文字数制限があるXでの投稿は、自然と要点を絞り込む必要があり、抽象化の良いトレーニングになります。学んだことを140〜280文字でまとめる練習をしましょう。」

8. **0円化を見抜く抽象フレーム**
   - テクノロジー×ボトルネック
   - 写真、AI、自動運転の例

9. **今すぐ使えるチェックリスト**
   - 目的を1行で（抽象）
   - 数値指標を3つ（具体）
   - 行うことは？（具体）

10. **まとめ**
    - 具体＝行動、抽象＝法則
    - トレーニング方法の再確認
    - 参考文献

## 🎨 デザインの特徴

- **カラフルで親しみやすいUI**: グラデーション効果、カード型レイアウト
- **レスポンシブデザイン**: スマホ、タブレット、PC対応
- **アニメーション効果**: フェードイン、ホバー効果
- **アイコン**: Font Awesome 6使用
- **フォント**: Noto Sans JP（読みやすい日本語フォント）

## 🚀 使い方

### インストール
1. `index.html` をダウンロード
2. ブラウザで開く
3. インストール不要、すぐに使用開始！

### 動作環境
- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- インターネット接続（CDN経由でライブラリを読込）
- JavaScript有効

### オフライン使用
初回アクセス後、LocalStorageにデータが保存されるため、ブレインストーミング内容やハイスコアはオフラインでも利用可能です。

## 📦 技術スタック

- **HTML5**: セマンティックな構造
- **CSS3**: グラデーション、アニメーション、レスポンシブデザイン
- **JavaScript (ES6+)**: バニラJS（フレームワーク不使用）
- **外部ライブラリ（CDN経由）**:
  - Font Awesome 6（アイコン）
  - Google Fonts（Noto Sans JP, Roboto）
- **データ保存**: LocalStorage

## 📊 データ構造

### ゲーム状態
```javascript
gameState = {
    playerCount: 4,          // 参加人数
    turnCount: 1,            // ターン数
    difficulty: 'easy',      // 難易度（easy/hard/expert）
    currentPlayer: 0,        // 現在のプレイヤー
    currentRound: 0,         // 現在のラウンド
    currentTurn: 1,          // 現在のターン
    totalRounds: 0,          // 総ラウンド数（人数×ターン数）
    players: [],             // プレイヤー情報
    currentWords: [],        // 現在の単語
    currentAnswer: '',       // 現在の回答
    ratings: [],             // 評価の配列
    ratingCount: 0          // 評価数
}
```

## 🐛 既知の制限事項

1. **単語数**: 現在1,300語。10万語には拡張が必要
2. **評価システム**: 現在は順番に評価。同時評価には別の実装が必要
3. **モバイル**: ドラッグ&ドロップはマウス操作のみ対応（タッチ対応は今後追加予定）
4. **オフライン**: 初回アクセス時はインターネット接続が必要（CDNライブラリ読込のため）
